Interaction Network



Interaction Network is a graphic tool to show the interaction network of Post-Translational Modifications.


HOW TO ACCESS:


Users can use the Interaction Network Search in the Network page. For example, you can browse the network for 'p53' and 'H3', as shown below. Try browsing p53,H3.




USAGE: Users can see the results including two parts. One is the Interaction Network between target protein and enzyme, the other is Distribution of the target protein of PTMs. The target protein will be the center of the network, and the nodes are the interacted enzymes. In the network, users can control the distance of protein-enzyme interactions in the network by the controller in the lower right corner.
